# macOS

## Download

[Download AeonSage for macOS](https://aeonsage.org/downloads/AeonSage-mac.dmg)

## Install

1) Open the DMG.
2) Drag **AeonSage** into **Applications**.
3) Launch **AeonSage** from Applications.

## First run

- The app launches the local configuration UI automatically.
- If it does not open, click the menu bar icon and choose **Settings**.

## Permissions

macOS may ask for permissions (accessibility, screen recording) depending on which
tools and channels you enable. Grant only what you use.
