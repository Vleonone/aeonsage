# Windows

## Download

[Download AeonSage for Windows](https://aeonsage.org/downloads/AeonSage-win.exe)

## Install

1) Run the installer.
2) Complete the setup wizard.
3) AeonSage launches automatically after install.

## First run

- The app opens the local configuration UI automatically.
- If it does not open, launch **AeonSage** from the Start menu.
